# 🌴 Sri Lanka Tourism Web Application

## 📌 Overview

Sri Lanka, known as the **“Pearl of the Indian Ocean,”** is a country rich in natural beauty, culture, and history.
This project is a tourism web application designed to showcase the best travel destinations, experiences, and cultural highlights of Sri Lanka.

It helps users explore popular locations, plan trips, and learn about the country in an interactive and user-friendly way.

---

## ✨ Features

* 🗺️ Explore top tourist destinations
* 🏝️ Beach, hill country, and cultural site highlights
* 📖 Detailed descriptions of locations
* 🔍 Easy navigation and user-friendly interface
* 📱 Responsive design (mobile & desktop)
* 🌐 Travel inspiration and recommendations
